# WildSight / 识野

**让好奇识得万物**

WildSight 是一款面向自然观察与旅行探索场景的 AI 识别 App。用户可以上传动物的图片或视频，识别它是谁，了解它的外形特征、行为习性和有趣冷知识，并把每一次遇见保存成一张手绘风的生物身份卡。

## 产品理念

WildSight 想解决的问题不是“查一个百科词条”，而是让用户愿意走近自然、认识生命。

传统动物识别很容易变成冷冰冰的结果查询：名字、拉丁名、分类、分布。但真实的探索体验更像是一次遇见。用户可能是在潜水时看到一条鱼，在路边遇到一只鸟，或者在旅行中拍下一段陌生动物的视频。WildSight 希望用更轻松、有趣、容易读懂的科普内容，把这次遇见变成一次新的认识。

产品核心理念：

- 对世界永远保持好奇和热情。
- 不停探索新的环境，遇见新的生命。
- 把每一次奇妙相遇都记录下来。

## 设计思路

当前版本重点验证三个核心能力：

1. **图片 / 视频识别**  
   支持上传动物图片或视频。视频会先抽取关键帧，再交给识别服务判断主体。

2. **轻科普内容生成**  
   识别后生成更适合普通用户阅读的内容，包括“它是谁”“怎么生活”“冷知识”等信息。内容风格避免百科式堆砌，强调清晰、轻松和有趣。

3. **手绘生物身份卡**  
   根据用户上传的动物图片生成一张手绘风身份卡。身份卡会和科普内容一起保存到记录中，形成用户自己的自然观察收藏。

后续计划支持：

- 按地点回顾见过的动物。
- 按类目、时间、旅行等维度整理身份卡。
- 为同一种动物聚合多次遇见记录。
- 支持更完整的自然观察收藏册。

## 项目结构

- `WildSight-iOS/`：SwiftUI iOS App。
- `server/`：识别与手绘图生成服务。
- `MiniDemo.html`：可放入简历的网页体验版。
- `prompt/`：识别、科普文案与手绘图生成相关提示词。
- `docs/`：产品与接口文档。
