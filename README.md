# 奇遇簿小程序 demo

这是一个记录自然奇遇的微信小程序 MVP demo，当前重点验证核心交互闭环：

1. 从相册上传图片或视频。
2. 视频超过 10 秒时进入片段选择页。
3. 生成 mock 识别结果并自动加入图鉴。
4. 在图鉴中查看手绘风卡片。
5. 进入详情页查看原照片/原视频入口，或从右上角更多菜单删除记录。

## 当前 mock 的部分

- AI 识别结果是本地 mock，暂未接真实模型。
- 视频超过 10 秒后的“裁剪”目前保存片段起止时间，未真正生成裁剪后的视频文件。
- 图鉴插画是 CSS 生成的手绘贴纸风占位，后续可以替换为 AI 生成图或静态资源。
