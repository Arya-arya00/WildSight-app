# 识野 iOS Demo

这是“识野”的 SwiftUI iOS 前端 Demo。

## 打开方式

用 Xcode 打开：

`WildSight-iOS/WildSight.xcodeproj`

## 当前已实现

- 记录首页：点击上传入口后弹出选择面板，再从相册选择图片或视频。
- 图片识别：选择图片后生成一条 mock 识别结果，并自动进入结果页。
- 视频识别：读取视频时长；超过 10 秒进入片段选择页，短视频直接生成结果。
- 结果页：展示手绘风图鉴卡、时间地点、小科普内容；文字可直接点击修改。
- 图鉴页：以卡片墙展示所有记录。
- 详情页：展示生成图鉴图；原始照片/视频从右下角入口查看；删除记录藏在右上角菜单中。
- 编辑页：支持修改名称、分类、简介、科普段落、时间和地点。

## 当前还是 mock 的部分

- AI 识别结果还没有接真实接口。
- 手绘图鉴图目前是 SwiftUI 生成的示意插画，不是根据识别对象动态生成。
- 视频超过 10 秒时目前保存片段起止时间，还没有真正导出裁剪后的视频文件。
- 数据暂存在内存里，下一步可接本地持久化或 iCloud。
