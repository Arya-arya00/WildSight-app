# 手绘图生成 Prompt

把用户上传的动物照片，改绘成一张适合自然图鉴卡片使用的可爱手绘贴纸。

## 参考风格

- 参考“海洋图鉴样式.jpeg”的感觉：可爱的海洋动物贴纸合集风格。
- 软水彩 / gouache 上色，颜色柔和，有轻微渐变。
- 造型圆润、偏胖一点，但不能丢掉物种关键识别特征。
- 线条是暖灰色或深灰色手绘描边，边缘略微不均匀，不要矢量图的死板边。
- 眼睛可以是小黑点，表情轻微可爱，但不要强行拟人。
- 背景保持白色或接近透明的干净底色。

## 主体要求

- 只画用户上传照片里的主体动物或自然对象。
- 尽量保留原照片里的姿态、朝向、身体角度和正在发生的动作。
- 如果原图能看到全身，就画全身；如果原图只看到局部，就在不编造关键结构的前提下补成自然完整的主体。
- 识别名称：{name}
- 分类提示：{latin}
- 观察特征：{summary}

## 构图限制

- 画面只保留一个居中的主体。
- 不要额外添加其他动物、贝壳、星星、泡泡、道具、文字、签名、边框或复杂背景。
- 不要照片写实，不要 3D，不要扁平矢量，不要人形动漫风。
- 4:3 横向身份卡尺寸，圆角卡片里展示要自然，干净、卡片可用。

## 人物照片彩蛋规则

如果识别结果是“人类朋友”，手绘图不是自由创作，而是把用户上传的原始人物照片改绘成身份卡插画。

- 必须以原图为准，不要想象另一个人。
- 尽量保留人物主体的姿态、朝向、身体角度、手势、衣着颜色、配饰、手里拿的物品和整体构图。
- 如果原图里人物在笑，就保留可见笑容；如果神情安静或中性，就保留这种温和表情，不要强行改成大笑或悲伤。
- 不要把人物美化成通用模特、明星、动漫角色，也不要改变年龄感。
- 如果基于原图编辑失败，不要退回纯文字生成人物图，宁可不生成身份卡。
- 风格仍然是可爱手绘贴纸 / 身份卡插画，软水彩或 gouache 上色，暖灰手绘描边，干净背景，4:3 横向身份卡尺寸。
